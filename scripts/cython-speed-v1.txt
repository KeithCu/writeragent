Column guide:
  direction     ingress = Host -> Child (LibreOffice Host sending data into the sandboxed Child worker).
                egress = Child -> Host (sandboxed Child worker returning data back to the LibreOffice Host).
  wire_format   json_list = nested floats in JSON (slow materialize).
                split_grid = compact base64 float64 with sparse strings in JSON (fast materialize).
                pickle5 = Pickle protocol 5 with nested lists (force=never; slow np.array materialize).
                pickle5+sg = Pickle protocol 5 with split_grid envelope (force=always; fast frombuffer).
  wire_KiB      Line/payload size on the wire for this row.
  vs_json_wire  On comparing rows: size vs paired json_list row (same shape/direction). json_list row shows 'baseline (json)'.
  materialize   timings for rebuilding data: ingress (child np.array or frombuffer) / egress (host unpack to lists).
  mat_x, total_x, e2e_faster  Shown only for speedups relative to the baseline.
  mat_x         Speedup factor (e.g. 4.74x): json_list materialize / format materialize.
  total_x       Speedup factor (e.g. 2.06x): json_list total_ms / format total_ms.
  e2e_faster    wire_format that won on total_ms for this shape.
